# EventFinder - Featured Events Webpage

## Objective
The goal of this project is to **build a responsive, interactive webpage** that displays a list of upcoming events in a clean, mobile-friendly layout.  
This project demonstrates my ability to implement modern web development practices using HTML, CSS, and JavaScript.

---

## Technologies and Tools Used

- **HTML5**: Semantic structure for sections, navigation, and event cards
- **CSS3**: Styling for a colourful, clean, and responsive UI
  - Used CSS Grid and Flexbox for layout
  - Media queries for mobile and tablet responsiveness
- **JavaScript (Vanilla JS)**: 
  - Dynamic rendering of events from a JSON array
  - Search bar to filter events by name
  - Smooth scroll for navigation
- Optional: **VS Code Live Server** for local testing

---

## Features

1. **Navigation Bar**
   - Contains site logo and links to Home, Events, and Contact
   - Responsive hamburger menu on mobile devices

2. **Hero Section**
   - Heading: “Discover Events Near You”
   - Short description

3. **Featured Events Section**
   - Displays dynamic event cards (11 sample events included)
   - Each card shows:
     - Event name
     - Date and time
     - Location
     - Short description
     - “View Details” button linking to event detail page

4. **Event Detail Page**
   - Each event has a separate page showing full information dynamically based on the event ID

5. **Search Functionality**
   - Users can filter events by name in real time

6. **Contact Section**
   - Simple contact form with name, email, and message
